# 🛍️ Customer Segmentation & Product Recommendation System

## 📌 Project Overview

This project is an AI-powered Customer Segmentation and Product Recommendation System developed using Machine Learning techniques on Online Retail data.

The system performs:

* Customer Segmentation using RFM Analysis and K-Means Clustering
* Product Recommendation using Product Similarity Analysis
* Interactive Dashboard using Streamlit

The application helps businesses understand customer behavior and recommend similar products to improve customer engagement and sales.
